# EJS Plugin for DocPad
Adds support for the [EJS](https://github.com/visionmedia/ejs) templating engine to [DocPad](https://docpad.org)

Convention:  `.anything.ejs`


## Install

```
npm install --save docpad-plugin-ejs
```
